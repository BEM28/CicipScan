# 🍲 CicipScan: AI-Powered Food Intelligence

CicipScan is a premium Flutter application that leverages Artificial Intelligence to identify food items and provide comprehensive nutritional insights. Using a custom TensorFlow Lite model and the Google Gemini API, CicipScan transforms your camera into a personal nutritionist.

---

## ✨ Features

- **🔍 Intelligent Scan**: Real-time food recognition using a high-performance TFLite model.
- **🖼️ Smart Cropping**: Manual cropping tool to ensure precise focus on food items for better accuracy.
- **🧬 AI Nutritional Analysis**: Detailed breakdown of calories, macronutrients (Protein, Carbs, Fat), and essential vitamins.
- **💡 Gemini AI Insights**: Dynamic, contextual health insights generated by Google's Gemini models.
- **👨‍🍳 Recipe Integration**: Integration with TheMealDB API to provide recipe names, areas, and assembly instructions.
- **📜 Scan History**: Local database to keep track of your previous culinary discoveries.
- **🎨 Premium UI**: Modern design featuring smooth Lottie animations, glassmorphism, and a sleek dark mode.

---

## 🛠️ Tech Stack

- **Framework**: [Flutter](https://flutter.dev/) (Material 3)
- **AI/ML Engine**: TensorFlow Lite ([tflite_flutter](https://pub.dev/packages/tflite_flutter))
- **LLM Integration**: [Google Gemini API](https://ai.google.dev/)
- **State Management**: [Provider](https://pub.dev/packages/provider)
- **Database**: [SQLite](https://pub.dev/packages/sqflite)
- **Networking**: [HTTP](https://pub.dev/packages/http)
- **Assets**: 
  - [Lottie](https://airbnb.design/lottie/) for high-quality animations.
  - Custom trained MobileNetV2 TFLite model.

## 🚀 Getting Started

### Prerequisites

- Flutter SDK (>= 3.10.0)
- Android Studio / VS Code
- A valid Google Gemini API Key

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/cicipscan.git
   cd cicipscan
   ```

2. **Setup Environment Variables**
   Create a `.env` file in the root directory and add your API keys:
   ```env
   GEMINI_API_KEY=your_actual_api_key_here
   GEMINI_MODEL=gemini-1.5-flash
   ```

3. **Install Dependencies**
   ```bash
   flutter pub get
   ```

4. **Run the App**
   ```bash
   flutter run
   ```
